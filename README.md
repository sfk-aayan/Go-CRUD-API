# Go-CRUD-API

## Dependencies
- GORM
- Go Gin
- MySQL
- GoDotEnv

## Installation
### GORM
```powershell
go get -u gorm.io/gorm
go get -u gorm.io/driver/mysql
```
### Go Gin
```powershell
go get -u github.com/gin-gonic/gin
```
### GoDotEnv
```powershell
go get github.com/joho/godotenv
```
